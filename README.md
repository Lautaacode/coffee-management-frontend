## 🧠 Descripción General

El sistema de gestión para la cafetería “Buen Sabor” es una aplicación
diseñada para digitalizar, centralizar y optimizar los procesos operativos del negocio.
La aplicación actúa como un servidor centralizado, donde se gestionan las distintas operaciones del establecimiento,
incluyendo la administración de usuarios con roles específicos (gerente, cajeros, meseros y cocina),
el registro de pedidos (comandas), el control de ventas y la generación de información útil para la toma de decisiones.
El objetivo principal es transformar la operación de “Buen Sabor” en un flujo de trabajo ágil,
confiable y basado en datos, mejorando tanto la experiencia del cliente como la eficiencia interna del equipo.
